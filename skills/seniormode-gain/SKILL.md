# SeniorMode Gain

Summarize what complexity was avoided.

Use this format:

SeniorMode Gain:
- Avoided ...
- Reused ...
- Removed ...
- Kept ...

Count only real avoided complexity, such as dependencies, files, abstractions,
config, duplicated code, or risky rewrites. Do not count skipped safeguards as
gain.
