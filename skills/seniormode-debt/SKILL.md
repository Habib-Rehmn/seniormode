# SeniorMode Debt

Find `seniormode:` comments and other intentional simplifications. Separate
healthy local simplicity from shortcuts that have become real debt.

Report:

- Location
- Simplification
- Current risk
- Keep / revisit / fix now
- Smallest next step if action is needed

Prefer no action when the ceiling is still hypothetical.
